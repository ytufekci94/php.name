PHP Arrays

An array in PHP is a data structure that stores a collection of values. Arrays can store values of any data type, including integers, strings, and objects.

There are two types of arrays in PHP: indexed arrays and associative arrays.

An indexed array is a simple list of values, where each value is assigned a numerical index. The index begins at 0 and increases by one for each element in the array. Indexed arrays are created using the array() function, like this:

Example

<?php

$colors = array('red', 'green', 'blue');

?>

You can access the elements of an indexed array using the square bracket notation, like this:

Example

<?php

echo $colors[0]; // Outputs 'red'
echo $colors[1]; // Outputs 'green'
echo $colors[2]; // Outputs 'blue'

?>

An associative array is an array that uses string keys to access its elements. Associative arrays are created using the array() function, like this:

Example

<?php

$ages = array(
  'John' => 35,
  'Jane' => 27,
  'Bob' => 43
);

?>

You can access the elements of an associative array using the square bracket notation and the key, like this:

Example

<?php

echo $ages['John']; // Outputs 35
echo $ages['Jane']; // Outputs 27
echo $ages['Bob']; // Outputs 43

?>

PHP also provides several built-in functions for working with arrays. The count() function is used to get the number of elements in an array, the sort() function is used to sort the elements of an array in ascending order, and the rsort() function is used to sort the elements of an array in descending order.

There are many other functions for working with arrays in PHP, including functions for adding and removing elements, searching for elements, and manipulating the elements in various ways.
