PHP Switch

The switch statement is a control flow statement that is used to execute different code blocks based on multiple conditions. It is similar to the if-elseif-else statement, but is often used when there are many conditions to check.

Example

<?php

switch (expression) {
  case value1:
    code to be executed if expression == value1;
    break;
  case value2:
    code to be executed if expression == value2;
    break;
  ...
  default:
    code to be executed if expression does not match any of the values;
}

?>

The expression in the switch statement is evaluated, and the value is compared to each of the values in the case statements. If a match is found, the code block associated with that case is executed. If no match is found, the code in the default block is executed.

It is important to use the "break" statement at the end of each case block, as it prevents the code from falling through to the next case block. If you do not use the "break" statement, all of the code blocks below the matching case block will be executed, even if they do not match the expression.

Example

<?php
$x = 10;

switch ($x) {
  case 1:
    echo "x is 1";
    break;
  case 10:
    echo "x is 10";
    break;
  default:
    echo "x is something else";
}

?>

In this example, the value of $x is 10, so the code block associated with the "case 10:" statement is executed, and the output is "x is 10".
