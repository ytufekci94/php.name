PHP MySQL Where

The WHERE clause is used to filter rows in a SELECT, UPDATE, or DELETE statement in MySQL. It specifies the conditions that rows must meet to be included in the result set or affected by the statement.

To use the WHERE clause in MySQL using PHP, you can use the mysqli_query() function with a SELECT, UPDATE, or DELETE statement that includes the WHERE clause.

Here is an example of a SELECT statement with a WHERE clause that retrieves rows from the customers table where the city column is 'New York':

<?php
$query = "SELECT * FROM customers WHERE city = 'New York'";

$result = mysqli_query($conn, $query);

if (!$result) {
    die('Error executing query: ' . mysqli_error($conn));
}

?>

You can also use the WHERE clause to filter rows based on multiple conditions using logical operators such as 'AND' and 'OR'.

<?php
$query = "SELECT * FROM customers WHERE city = 'New York' AND state = 'NY'";

$result = mysqli_query($conn, $query);

if (!$result) {
    die('Error executing query: ' . mysqli_error($conn));
}

?>

This statement retrieves rows from the 'customers' table where the city is 'New York' and the state is 'NY'.

You can also use the 'WHERE' clause to filter rows using comparison operators such as '=', '>', '<', '>=', '<=', and '<>'.

<?php
$query = "SELECT * FROM customers WHERE age > 30";

$result = mysqli_query($conn, $query);

if (!$result) {
    die('Error executing query: ' . mysqli_error($conn));
}
?>

This statement retrieves rows from the 'customers' table where the 'age' column is greater than 30.

The WHERE clause is a powerful tool that allows you to filter rows in MySQL using PHP. You can use it to retrieve specific rows or perform operations on a subset of the data in your database.
